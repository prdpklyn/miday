## AI Model Setup

This project uses the Gemma 3 1B model for on-device intelligence. The model file is large and is not included in the git repository.

1. Download the `gemma3-1b-it-int4.task` model file.
2. Place it in the `assets/models/` directory.
   - Path: `assets/models/gemma3-1b-it-int4.task`
